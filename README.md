### Testing Env
```
Python 3.10.10
```

### Startup

```
cd path/to/DiscordBotWithGPT
```

```
python3 -m venv .
```

```
source bin/activate
```

```
pip install -r requirements.txt
```

```
python3 -m src.main
```
